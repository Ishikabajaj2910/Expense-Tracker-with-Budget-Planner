# 💰 Expense Tracker with Budget Planner

An intelligent and user-friendly Expense Tracker designed to help users efficiently manage their personal finances. This application allows users to record their income and expenses, set monthly budgets, monitor spending habits, and gain financial insights through interactive charts and reports.

---

## 📌 Overview

Managing personal finances can be challenging, especially when expenses are not tracked properly. This project provides a simple yet powerful solution for tracking daily transactions and planning budgets effectively.

The Expense Tracker with Budget Planner enables users to:

- Record income and expenses
- Categorize transactions
- Set monthly spending limits
- Visualize financial data
- Monitor savings and spending patterns
- Make informed financial decisions

---

## ✨ Features

- ➕ Add income and expense transactions
- 🗑️ Delete transactions
- 🏷️ Categorize expenses
- 📅 Track daily, weekly, and monthly expenses
- 💰 Set monthly budget goals
- 📊 Interactive charts and spending analytics
- ⚠️ Budget limit alerts
- 🔍 Search and filter transactions
- 💾 Data persistence using Local Storage
- 📱 Fully responsive design
